# 레거시 폴 개선 프로토타입

`quicksurvey/`, `quicksurvey-builder/`와는 별도 폴더. 기획팀 PM 확인(2026-06-22, [PROJECT_NOTES.md](../PROJECT_NOTES.md) 참고)에 따라
이번 프로젝트의 실제 방향은 **오픈서베이 수준의 자체 설문 빌더가 아니라, 기존 레거시 뉴스 폴 기능의 고도화**다.

이 폴더는 그 방향에 맞춰 "레거시 폴을 이렇게 개선하면 좋겠다"는 아이디어를 독립 데모로
미리 만들어보는 선행개발 공간이다. 실제 레거시 코드는 이 저장소에 없으므로, 여기서는
개선안을 시각적으로 검증할 수 있는 독립 프로토타입만 다룬다.

## 실행 방법

```bash
npm run dev
```

`http://localhost:5190` 접속.

아직 구체적인 개선 아이디어가 들어가지 않은 빈 스캐폴드 상태.
